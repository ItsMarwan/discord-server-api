Discord Lookup API

Simple, safe, and bot-powered user lookups for your Discord server

üîπ How it Works

This API requires a Discord Bot to be in the server you want to use it with. When you request user information:

  • The bot must already be invited to the guild (server).
  • The user you want to lookup must be in that same guild.
  • You must include the guild ID in your request to ensure the lookup is scoped only to your community.

This prevents stalking random users across Discord ‚Äî only server members can be queried.
